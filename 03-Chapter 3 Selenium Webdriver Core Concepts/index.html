üîπ 3.1 Working with Web Elements

‚úÖ find_element() and find_elements()

These are the most commonly used Selenium methods for locating HTML elements.

  • find_element(by, value) ‚Üí Returns one WebElement
  • find_elements(by, value) ‚Üí Returns a list of WebElements
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://example.com")

# Single element
username_input = driver.find_element(By.ID, "username")

# Multiple elements
all_links = driver.find_elements(By.TAG_NAME, "a")

üîπ Why needed?

You can‚Äôt interact with a page unless you can locate the HTML element.

‚úÖ click(), send_keys(), text, get_attribute()

  • click() ‚Äì Simulates a mouse click
  • send_keys() ‚Äì Types into an input box
  • .text ‚Äì Gets the visible text of an element
  • get_attribute() ‚Äì Gets the value of any HTML attribute
username_input.send_keys("admin")
login_button = driver.find_element(By.ID, "submit")
login_button.click()

üîπ 3.9 Waits: Implicit & Explicit

‚úÖ Implicit Wait

driver.implicitly_wait(10)

‚úÖ Explicit Wait

from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

wait = WebDriverWait(driver, 10)
wait.until(EC.presence_of_element_located((By.ID, "username")))

üîπ 3.12 Working with ActionChains

from selenium.webdriver.common.action_chains import ActionChains

actions = ActionChains(driver)
element = driver.find_element(By.ID, "hoverMenu")

actions.move_to_element(element).perform()       # Hover
actions.context_click(element).perform()         # Right click

# Drag and Drop
source = driver.find_element(By.ID, "dragMe")
target = driver.find_element(By.ID, "dropHere")
actions.drag_and_drop(source, target).perform()
